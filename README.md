# E-Commerce-Data-Analysis-TradeZone
📝 Project Overview
TradeZone is an expanding e-commerce platform that experienced significant regional growth in 2024. This project involves a full-scale audit of their 2023–2024 operational data. The goal of this analysis was to clean raw database extracts, resolve schema inconsistencies, and generate actionable insights regarding customer acquisition, seller performance, and revenue concentration.

🛠️ Technical Skills Demonstrated
Data Engineering & Cleaning: Standardized text formatting, handled NULL values mathematically, and executed safe deduplication using CTID and subqueries.

Advanced SQL: Utilized Common Table Expressions (CTEs), multi-table JOINs, conditional filtering (FILTER WHERE), and date-time interval math to prevent join-explosions .

Business Strategy: Translated raw query outputs into an Executive Analyst Memo delivering strategic recommendations to the Head of Growth and Head of Seller Operations.

💡 Key Insights
Conversion Gaps: Lagos and the FCT drive the highest acquisition, but require targeted "First-30" incentives to improve the 30-day conversion rate.

Seller Impact: Identified a direct correlation between slow seller fulfillment (48+ hours) and "Low Rated" product categories, directly impacting premium pricing strategies.
